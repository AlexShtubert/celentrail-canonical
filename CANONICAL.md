# Celentrail — КАНОНИЧЕСКАЯ СПЕЦИФИКАЦИЯ (живой документ)

**Назначение:** это единственный источник правды о системе Celentrail. 
AI_CONTEXT: data/AI_CONTEXT.json
Ссылка на репозиторий передаётся любому ИИ, чтобы он **сразу понимал всё**: геометрию, кинематику, нагрузки, материалы, техпроцесс, допуски, версии, экспорт 3D.

## 0) Как пользоваться (для любого ИИ)
1) Сначала прочитать: `docs/00_MASTER_CONTEXT.md`
2) Затем: `docs/01_SYSTEM_OVERVIEW.md`
3) Затем узлы:
   - `docs/10_BALKA.md` (направляющая / балка)
   - `docs/20_TROLLEY.md` (тележка / роликовый блок)
   - `docs/30_TRACK_AND_ROLLERS.md` (дорожка и ролики)
   - `docs/40_LOADS_AND_TARGETS.md` (нагрузки, цели по усилиям)
4) Для геометрии всегда использовать **актуальные файлы** в `assets/balka/latest/`.

## 1) Термины
- **Celentrail** — название системы.
- **Балка** — консольная направляющая (актуальная геометрия: `assets/balka/latest/`).
- **Тележка** — настенный кронштейн/плита + валы + ролики (внутри балки).
- **Дорожка** — рабочая поверхность контакта с роликом (V-кинематика).
