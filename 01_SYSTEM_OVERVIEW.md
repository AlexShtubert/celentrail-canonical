# Celentrail — обзор системы (главное, чтобы не перепутать кинематику)

Celentrail — скрытая консольная система раздвижной двери.

✅ **Кинематика (ВАЖНО):**
- **Тележка (роликовый блок) закреплена на стене и НЕ перемещается.**
- **Перемещается “балка” вместе с дверью** — балка проходит через ролики тележки.
- То есть: **НЕ “тележка едет по балке”**, а **“балка ездит по тележке”.**

При этом ролики заходят внутрь профиля балки, поэтому снаружи почти ничего не видно — остаётся “чистая” дверь.
