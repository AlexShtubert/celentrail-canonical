# 40_LOADS_AND_TARGETS — нагрузки и цели

## Нагрузочная модель (зафиксировано)
- Дверь + балка ~60 кг
- Вылет центра массы x ≈ 575 мм
- База роликов ≈ 260 мм
- Момент M ≈ 339 Н·м

### Ролики (крайние положения)
- Закрытие влево: левый верхний ~1300 Н + правый нижний ~713 Н
- Открытие вправо: правый верхний ~1300 Н + левый нижний ~713 Н
- Верхние ролики — главные, нижние — прижим/поддержка

## Цели по усилию
- Ход: 4–8 Н
- Старт: 10–18 Н
